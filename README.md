# 宠物领养管理系统 java1123

## 项目概述

宠物领养管理系统是一款全功能开源系统，旨在为用户提供一站式的宠物领养、救助、捐赠以及论坛互动等功能。系统基于成熟的技术框架，具有稳定性高、易用性强、适用范围广的特点。

## 技术栈

### 架构模式

- B/S（浏览器/服务器）
- MVC分层设计

### 开发环境

- 后端开发工具：IDEA
- 前端开发工具：VS Code
- JDK版本：1.8
- 依赖管理：Maven
- 数据库：MySQL

### 核心框架

- 后端：SpringBoot + MyBatis
- 前端：Vue.js + Element UI

### 部署支持

- Docker容器化部署（可选）

## 功能模块

### 系统角色

#### 管理员权限

- 用户管理：注册审核、角色分配、权限控制
- 宠物管理：流浪动物信息录入、绝育状态跟踪、领养申请审核
- 救助管理：救助站信息维护、喂养点地图定位、走失宠物报备
- 内容管理：公告发布、科普文章编辑、活动策划与报名
- 数据统计：领养率、捐赠金额、论坛活跃度等数据分析

#### 用户功能

- 首页：轮播图、热门活动、快速领养入口
- 宠物领养：分类筛选（品种、年龄、性别）、在线申请
- 流浪救助：上报流浪动物位置、查看救助进度
- 论坛互动：发帖、评论、点赞、私信交流
- 捐赠支持：物资/资金捐赠记录、透明化公示

### 运行环境

- 支持Windows/Mac系统
- 兼容主流浏览器
